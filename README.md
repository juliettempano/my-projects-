## **PMDD Symptom Severity Analysis** ##

**Project Overview**

This project explores relationships between stress levels, lifestyle factors, age groups, and symptom severity in women’s health data related to PMDD and mood disorders.

**Objectives**

- Analyze symptom severity trends
- Explore age-based differences
- Investigate relationships between stress and lifestyle factors
- Practice SQL-based exploratory analysis

**Tools Used**

- SQL
- Excel
- Git/GitHub

**Key Analyses**

- Stress levels vs average symptom severity
- Lifestyle factors by age group
- Treatment history comparisons
- PMDD subgroup analysis
  
**Key Findings**

- Higher stress levels were generally associated with increased symptom severity
- Respondents aged 30–35 reported consistently high severity across categories
- Lifestyle patterns showed inconsistent trends, suggesting possible confounding factors

**Future Improvements**

- Add Tableau dashboard
- Perform statistical significance testing
- Expand analysis with additional datasets
